Webringie Demo

See how easy it is to integrate your site with Webringie!

Adding Your Site

To add your site, create a metadata.json file and place it at the root of your site. Here’s a sample:

{
    "website": "https://yourwebsite.com",
    "title": "Your Website Title",
    "description": "A brief description of your website.",
    "keywords": ["keyword1", "keyword2", "keyword3"],
    "backlink": "https://yourwebsite.com/backlink"
}

Step 1: Set Up Your Metadata

Make sure your metadata.json file is correctly configured and accessible.

Step 2: Add the Script Tag

Add the following line to your HTML:

<script src="https://juliusthejules.github.io/webringie/js/webringie.js" defer></script>
