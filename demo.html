Webringie Demo

See how easy it is to integrate Webringie into your website!

Demo Integration

To see Webringie in action, follow these steps:

  1. Include the following script tag in your HTML:
    2. <script src="https://juliusthejules.github.io/webringie/js/webringie.js" defer></script>
  2. Create a metadata.json file with your site's metadata.
  3. Once set up, your site will automatically be linked to others in the Webringie network.

Example Metadata File

{
    "siteName": "Your Site Name",
    "siteURL": "https://yourwebsite.com",
    "description": "A brief description of your site.",
    "keywords": ["SEO", "Webringie", "metadata"]
}

Try it out and see how it boosts your visibility!
